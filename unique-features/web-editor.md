# ⚕️ WEB Editor

### Edit your language files with a WEB UI

1. /rediseconomy editmessage \<message>
2. Click on this message:\
   ![](<../.gitbook/assets/image (7).png>)
3. The WEB-Editor now opens in the standard browser
